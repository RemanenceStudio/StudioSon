Installation New theme installation

× Attention! You need to have a running WordPress before you install your theme. If you don't have any installed WordPress, please follow this link http://codex.wordpress.org/Installing_WordPress

You need to start your installation by uploading your theme. Then you need to activate it. There are 2 ways of uploading a theme. Uploading over WordPress engine or Using an FTP software.

WordPress Upload

In your control panel, go to Apperance > Add New Themes > Upload page. Click toBrowse and choose the zip file inside from your theme folder.Then click to Install Now to finis installation.

× Attention! If you get a message like "Your upload limit is not enough", please contact your host provider or switch to FTP upload.

FTP Upload

You need to unzip your theme files before uploading it by an FTP software. Then you need to upload your files to /wp-content/themes/

Then you need to activate your theme from Apperance > Themes page.

Important

After your theme activation you need to go to RB Admin page. Then, theme will automatically check all of your settings. If you have any error message on this page, you will find your solution in this help documentation.

Plugin Installation

This theme includes a customized plugin. There are 2 ways to upload a plugin. You need to activate the plugin after uploading. This theme uses "WP-Pagenavi" plugin for pagination of blog and portfolio pages. To install your plugin over WordPress, go to Plugins > Add New > Upload page and click to Browse button. Then you need to choose the zip file from /Plugins folder which you downloaded from Theme Forest. Then click to Install Now button. Or you need to unzip files from /Plugins folder and upload them by an FTP software to /wp-content/plugins/ folder. After uploading your plugin, you need to activate it from Plugins > Installed Plugins page.

Customizing the Menu

This theme includes a positioned menu.From Apperance > Menus page, create a menu and give it a name. Then you need to choose its name from Main Navigation box, it will be assigned.

Assigning a Home Page

You need to create a page before you assign a Home Page. To create a new page go toPages > Add New. You may give a name like "Home". Then click to Publish and your page will be published.

Settings > Reading / "Front Page Displays" area / check A static page (select below) and select the page that we named as "Home" from listed at Front page:

Info: If you want your page looks like our Demo page, you can find demo_content.xml from your downloaded folder. Then you need to import it from WordPress. Or you can use demo home pages' codes from /Documentation / FrontPage.txt and FrontPageAlternative.txt

Creating a Page Customize Your Pages

You can start creating a page by Pages > Add New Page . After adding your title and content click to Publish then your page will be published. To see your page at your menu , you need to add your page to menu from Apperance > Menus page.

Sidebar Position

You can position your sidebar from RB Admin > GENERAL SETTINGS page with "Default Sidebar Position on Page" settings. It positions the sidebars for the pages. If you want a spesific position for your spesific page you should choose Right, Left or None from "Sidebar Option".

Info: You can find demo_contemt.xml file (in downloaded file) and by importing it to WordPress you can have a main page and contact page designs like our demo pages. And you can find the shortcodes of home page and comtact pages at /Documentation folder.

You can start adding new items to your portfolio by using Portfolios > Add New Item button.

First of all give your title and then under Format section (which is at the right part of the page) select the type like "Images", "Gallery" or "Video" then click to Publish. This process will show chosen format.

Preparing a Portfolio Page

You can use shortcodes instead of page templates for Portfolio, Blog or Gallery. It is much more flexible.

Create a page and use [portfolio] shortcode inside, its that simple to show your portfolio items in a single page.

Listing Type

Porttfolio can be listed in 2 ways. First, in pages with [portfolio type="pagination"] second is filtering with portfolio type="filter"].

If you want to use filter type, you need to seperate your portfolio items into categories.

Number of items on Page

If you use pagination type for your Portfolio it shows 10 items for a page. To change the number of items, you can use[portfolio type="pagination" postperpage="20"]shortcode.

Creating Multiple Portfolio

If you want to use multiple portfolio or if you want to show some categories, you can use [portfolio type="filter" category="1,2,3"] shortcode. You need to seperate category ids with comma.

Number of Columns

Default portfolio column number is 3. If you want to change it to 3,4,5 or 6 you can use [portfolio count="6"] shortcode.

Image Type

"landscape" is the default format type for Portfolio image items. You can change the type with using [portfolio imagetype="portrait"] or [portfolio imagetype="square"]

Gallery Publish your Video and Images

Create a Gallery

You can publish your gallery in posts, as portfolio items or standalone. On the other hand, slider shortcodes may use gallery items too.

You can create your first Gallery by clicking Add a New Gallery in RB Admin > GALLERY MANAGER page and giving its name.

Galleries are called by their "ID" numbers instead of their names.

Edit Your Galleries

You can use Edit Gallery button to add or remove gallery items. It lists the gallery items.

Adding Images to the Gallery

By clicking Image > SELECT FILES button over item list, you can select and upload how many images you want at once.

Adding Youtube files to Gallery

By clicking Video > Youtube button over item list you need to enter video ID. Then you need to enter its width and height values.You can finish the process by clicking Add Video button. Thumbnails of Youtube files are created automatically.

Adding Vimeo files to Gallery

By clicking Video > Vimeo button over item list you need to enter video ID. Then you need to enter its width and height values.You can finish the process by clicking Add Video button. Thumbnails of Vimeo files are created automatically.

Adding Self Hosted files to Gallery

By clicking Video > Vimeo button over item list you need to enter your file's full URL.Then you need to enter its width and height values. You can finish the process by clicking Add Video button. You can use Upload Thumbnail button to upload your video thumbnail.

Edit your Gallery Items

You can move your items by draging or you can change their orders. Then click Save Changes to complete the process.

Changing Item's thumbnail

By clicking [Upload Thumbnail] button you can upload a new thumbnail.

Caption and Description Fields.

These fields can be shown accordimg to the content which gallery is shown in. Don't forget to save after you change their "Caption" and "Description" fields.

Removine an Item

You need to click [Delete] button which is under thumbnail to remove an item.

Removing the Gallery

You need to click Remove button to remove a gallert. Don't forget! If you remove a gallery, all the contents of the gallery will be removed.

Preparing a Gallery Page

You can use shortcodes for Portfolio, blog or gallery instead of Page Template. It is much more flexible.

Create a page and use [gallery id="1"] shortcode. Its that simple to show your images and video files

Automatic Image Resizing

Gallery resized the thumbnails automatically. If you don't want automatic resizng you can use [gallery type="portrait" usetimthumb="false"]

Number of Columns

Default column number is 3. If you want, you can use 3,4,5 and 6 columns by using [gallery id="1" count="6"] shortcode.

Image Type

"landscape" is the default format for portfolio items. You can change it to "portrait" and "square" by using [gallery type="portrait"]

Translate MultiLanguage Support

How to support for languages

There are default.po and default.mo files in theme folder. You can edit them by using an editor for any language.

If you find a po editor, try poEdit

After saving your default.po and default.mo files, name it according to international code.

For example for Turkish tr_TR.po and tr_TR.mo

You need to set your WPLANG variable to display different languages in your WordPress configuration. For more information please visit here.
